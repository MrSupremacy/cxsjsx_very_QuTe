# 北京大学《程序设计实习》大作业项目 **SteveRun**
 
> 本项目是为北京大学《程序设计实习》课程开发的期末大作业。项目命名 `cxsjsx_very_QuTe`，既包含了课程的拼音首字母，也寓意着这是一个基于 **Qt (C++)** 开发的、充满趣味的项目。

## 💻 技术栈与运行环境

*   **开发语言**：C++ (C++11/14 及以上)
*   **图形界面框架**：Qt 6.x
*   **构建工具**：qmake
*   **支持系统**：Windows / macOS / Linux

---

## 🚀 编译与运行指南

为了方便助教和同学们在本地运行该项目，请参考以下步骤：

### 1. 克隆仓库
```bash
git clone https://github.com/MrSupremacy/cxsjsx_very_QuTe.git
cd cxsjsx_very_QuTe
```
### 2. 环境准备
请确保本地已安装 Qt Creator 及其对应的编译器套件（如 MinGW、MSVC 或 Clang），还有 Qt 包 multimedia 和 openglwidgets。
